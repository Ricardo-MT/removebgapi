# How To Use in macOS 
```
Clone the repo and cd to removebgapi-main
```
```
$python3.9 -m venv rmbg_venv
```
```
$source rmbg_venv/bin/activate
```
```
$pip install -r requirements.txt
```
```
$uvicorn main:app
```
